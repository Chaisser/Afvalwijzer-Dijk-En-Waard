## ✨ Belangrijkste Functionaliteiten

* 📅 **Persoonlijke Kalender:** Vul je postcode en huisnummer in en zie direct wanneer het PMD, GFT, Papier of Restafval wordt opgehaald.
* 🤖 **AI Afval Scanner:** Maak een foto van je afval en onze AI vertelt je direct in welke bak het thuishoort (max. 10 scans per maand).
* 🔔 **Slimme Notificaties:** Ontvang automatisch een herinnering de avond vóór de ophaaldag. Vergeet nooit meer de kliko buiten te zetten!
* ✈️ **Offline Modus:** Je kalender wordt lokaal opgeslagen. Zelfs zonder internetverbinding kun je je ophaaldagen bekijken.
* 🔒 **Privacy First:** Je adresgegevens worden alleen gebruikt om de kalender op te halen en worden niet gedeeld met derden.

## 🛠️ Voor Ontwikkelaars (Tech Stack)

Deze app is gebouwd met moderne web- en mobiele technologieën:
* **Frontend:** React Native (Expo)
* **Backend / API Proxy:** n8n (omzeilt geoblocking en optimaliseert API calls)
* **AI Vision:** OpenAI `gpt-4o-mini` via een beveiligde webhook.
* **Notificaties:** Expo Push Notifications & Local Scheduling.
